offregister_mesos
===============

[Apache Mesos](http://mesos.apache.org) is a multicluster manager that provides efficient resource isolation and sharing across distributed applications, or frameworks.

This package follows the [offregister](https://github.com/offscale/offregister) specification to setup and serve Mesos.

## Install dependencies

    pip install -r requirements.txt

## Install package

    pip install .
